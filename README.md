# Employee Salary Insights & Inequality Analysis using R

Objective
To analyze salary distribution and identify factors affecting employee compensation.

Tools Used
- R
- dplyr
- ggplot2

Steps Performed
- Data cleaning (handled missing values)
- Exploratory Data Analysis
- Correlation analysis
- Data visualization

Key Insights
- Salary increases with experience
- Some departments have higher salaries
- Salary distribution varies across employees
- Outliers exist in salary data

- analysis.R
